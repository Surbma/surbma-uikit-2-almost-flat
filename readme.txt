Some useful UIkit 2 styles for your site.

== Description ==

This plugin does only one thing: it enques some css and js files for you to use UIkit styles on your website. It does not load the whole UIkit framework, just some useful components. The css and js files are minified already to make it as light as possible.

UIkit theme: Almost Flat

UIkit Components included with this plugin:

- Button
- Dropdown
- Icon (Font Awesome)
- Modal
- Nav
- Panel
- Switcher
- Tab
- Table
- Utility

For more info about UIkit and all the components: <a href="http://getuikit.com/docs/components.html" target="_blank">UIkit Components Â»</a>

== Installation ==

1. Upload the `surbma-uikit-2-almost-flat` folder to the `/wp-content/plugins/` directory
2. Activate the Surbma - UIkit 2 - Almost Flat plugin through the 'Plugins' menu in WordPress
3. That's it. You can now use UIkit components on your site. (Not all of them!)
